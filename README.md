6. About (Short Description / Summary):	

R-Place Is a  App which has a aim of making a graphical view of entire Placement Process . Its Features Include charts for various parameters which help in success of entire training and Placement Procedure . 

7. Install (Installation procedure): 

Step 1: Install the APP by Using .apk File ( Required android Version 4.0.3 and above)
Step 2:Click on the R-Place Icon .
 Step 3: The App Opens and Rplace Logo Will Appear,Please Wait for some time
 Step 4:Now R-Place Dashboard will Appear.
Step 5:Add the Student Data By Clicking on the Add Button.
Step 6: Fill all the Fields. And Click on Submit
Step 7: Add at least 1 Student in each branch in order for the app to work 
correctly OR You can load Test Data by Navigating to :Settings> Load Test Data (This Option will add 2000 Records approx  in the Data Base)										
Step 8 : You Can View Graphs of Placement Data By Navigating  to View>View Statistics OR You Can View Student Data By Navigating To : View> View Data
